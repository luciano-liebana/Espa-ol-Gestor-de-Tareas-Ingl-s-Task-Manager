# Gestor de Tareas / Task Manager

## Descripción / Description
Gestor de tareas simple en Python. Permite agregar, ver, borrar y marcar tareas como hechas. Las tareas se guardan automáticamente en un archivo de texto.

Simple task manager in Python. Allows you to add, view, delete, and mark tasks as done. Tasks are automatically saved in a text file.

## Funcionalidades / Features
- Agregar tareas nuevas / Add new tasks
- Ver todas las tareas / View all tasks
- Borrar tareas / Delete tasks
- Marcar tareas como hechas / Mark tasks as done
- Guardado automático en archivo de texto / Automatic save to text file

## Cómo usar / How to use
1. Cloná o descargá el repositorio / Clone or download the repository
2. Ejecutá `tareas.py` con Python 3 / Run `tareas.py` with Python 3
3. Elegí una opción del menú / Choose an option from the menu
